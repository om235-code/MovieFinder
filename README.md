# MovieFinder

MovieFinder is a full-stack web application built for IT 340.

## Milestone 2 Features
- User registration
- User login
- Password hashing with bcrypt
- Frontend to backend communication
- Backend to MySQL communication
- Data stored securely in MySQL
- UFW firewall configured on all 3 VMs

## Files
- index.html
- login.html
- register.html
- style.css
- server.js
- package.json
- package-lock.json

## VM Setup
### Frontend VM
- Apache
- Port 80 open

### Backend VM
- Node.js + Express
- Port 3000 open

### Database VM
- MySQL
- Port 3306 open
